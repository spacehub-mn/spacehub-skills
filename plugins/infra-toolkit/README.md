# infra-toolkit

Infrastructure skills for Cloudflare, Prometheus, and Linux server ops.

## Install

```bash
/plugin marketplace add spacehub-mn/spacehub-skills
/plugin install infra-toolkit@spacehub-mn
```

## Skills

| Skill | What it covers |
|---|---|
| [`cloudflare-d1-ops`](skills/cloudflare-d1-ops/SKILL.md) | D1 migrations, Time Travel + R2 backups, debugging migration failures |
| [`cloudflare-workers-debug`](skills/cloudflare-workers-debug/SKILL.md) | Worker bundle size, memory limits, service bindings, IP forwarding |
| [`prometheus-alerts`](skills/prometheus-alerts/SKILL.md) | Prometheus + Node/Windows Exporter setup, Grafana alert rules |

See [CHANGELOG.md](CHANGELOG.md) for version history.
